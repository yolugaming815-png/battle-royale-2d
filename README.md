# Battle Royale 2D .io

Petit jeu battle royale vue du dessus, jouable directement dans le navigateur.

## Lancer

Ouvre `index.html` dans ton navigateur.

## Commandes

- `ZQSD`, `WASD` ou les fleches pour bouger
- Souris pour viser
- Clic gauche ou `Espace` pour tirer
- `R` pour recharger
- `1` a `4` pour choisir un slot de hotbar
- `Shift` pour esquiver
- Reste sur un objet au sol pour le ramasser

## Contenu

- 40 joueurs dont 39 bots
- Armes, munitions, boucliers et soins a ramasser
- Depart a la pioche
- Hotbar de 4 slots avec echange par slot selectionne
- Changement de slot avec `1` a `4` ou la molette
- Munitions separees par type d'arme : pistolet, mitraillette, pompe, assaut, sniper
- Coffres places sur les structures avec plusieurs drops
- Raretes d'armes : commun, ordinaire, rare, epique, legendaire
- Ramassage temporise : 1s munitions, 2s armes, 4s soins/boucliers
- Bots avec pathfinding et ramassage temporise
- Bots en phase loot au debut : arme + bouclier avant chasse, sauf riposte
- Maisons visitables avec portes, murs et coffres
- Rivieres, ponts, grandes maisons, champs et buissons
- Eau traversable avec ralentissement de nage
- Champs/buissons qui cachent les joueurs a distance
- Zone de danger qui se referme
- Mini-carte, compteur de survivants, score et fil d'eliminations
- Menu avec skins, boutique, profil, amis, rename et parametres
